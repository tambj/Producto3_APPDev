### 1. Instalar EXPO

npm install -g expo-cli


###  2. Empezar proyecto

expo init nombre-proyecto


### 3. Empezamos/lanzamos el proyecto 
npm start


### 4. Ver proyecto en el navegador

 expo start --web

instalará la dependencia react-native-web y podremos ver el proyecto en el navegador
